# FundEQ (Next.js)

Marketing site for FundEQ built with Next.js (App Router) and CSS Modules styling (no Bootstrap needed).

## Getting started

1) Install dependencies: `npm install`
2) Run dev server: `npm run dev`
3) Build for prod: `npm run build` then `npm start`

The main page composes modular sections from `components/` with per-section CSS modules in `styles/`. Global theme variables live in `app/globals.css`.
